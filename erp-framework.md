# 梅睿贸易进销存管理平台 — 方案框架

## 一、技术选型

| 项 | 选择 | 原因 |
|---|---|---|
| **形态** | 本地 Web 应用 | 浏览器打开，无需安装，局域网/本机即用 |
| **前端** | 纯 HTML + CSS + JS（单页应用） | 不需要后端服务器，打开就能用 |
| **数据存储** | 本地 JSON 文件（通过 Node.js 读写） | 比 SQLite 简单，数据可导出为 Excel |
| **后端（本机）** | Node.js 轻量服务 | 负责读写数据文件、生成报表 |
| **后续扩展** | 可一键部署到 GitHub Pages + 云存储 | 如需多人使用可升级 |

## 三、数据结构（核心表）

### 产品 Products
```
产品编号 | 一级产品线 | 二级 | 三级 | 产品系列 | 产品名称(中/英)
规格 | 材质 | 尺寸 | 净重 | 包装信息 | 供应商 | 安全库存
FOB 成本(￥) | FOB 美金价 | B2C售价(R$) | B2B售价(R$)
```

### 采购单 Purchase Orders
```
订单号 | 供应商 | 下单日期 | 预计到货 | 状态 | 明细(产品/数量/单价)
运费 | 杂费 | 总金额(￥) | 付款状态
```

### 库存 Inventory
```
产品编号 | 当前库存 | 在途数量 | 已预订数量 | 可用数量
仓库位置 | 上次盘点日期 | 安全库存阈值
```

### 销售订单 Sales Orders
```
订单号 | 客户 | 下单日期 | 交货日期 | 状态
明细(产品/数量/FOB价) | 总金额($) | 付款状态
```

### 出货单 Shipments
```
出货单号 | 订单列表(合并) | 客户 | 出货日期 | 船名航次
集装箱号 | 装货港 | 目的港 | ETD | ETA
明细(产品/数量/件数/体积/重量)
```

## 五、界面风格

- **操作区 + 数据表格** 清晰分区
